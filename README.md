# Install-guide screenshots (assets branch)

This is an **orphan branch** that contains only images. It shares no history
with `main`, so these PNGs never appear in the `main` working tree or in the
published pip package.

The project README references them via absolute `raw.githubusercontent.com`
URLs pinned to this branch, e.g.:

```
https://raw.githubusercontent.com/jumodada/Drissionpage-MCP-Server/assets/03-doctor.png
```

| File | Shows |
|------|-------|
| `01-install.png` | `pip install drissionpage-mcp` success |
| `02-version.png` | `drissionpage-mcp --version` |
| `03-doctor.png` | `drissionpage-mcp doctor` (all checks green) |
| `04-claude-code.png` | Claude Code `mcp_settings.json` config |
| `05-cursor.png` | Cursor `mcp.json` config |
| `06-codex.png` | Codex CLI `config.toml` config |
| `07-connected.png` | MCP client with tools loaded |
| `08-cursor-ui.png` | Cursor Settings UI: Tools & MCPs → New MCP Server |

## Regenerating

The terminal/editor cards (`01`-`07`) are generated by
`docs/render_screenshots.py` **on the `main` branch**. After regenerating,
copy the refreshed PNGs onto this branch root and push.

`08-cursor-ui.png` is a real GUI screenshot, not generated.
